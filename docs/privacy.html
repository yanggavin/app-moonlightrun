Privacy Policy

Effective date: February 11, 2026

Data We Collect

MoonlightRun collects location data only during active run sessions to calculate route, distance, pace, and workout history.

Permissions

Location authorization is required before a run can start. If permission is denied, run tracking cannot begin until permission is enabled in iOS Settings.

How We Use Data

We use run data only to provide app functionality, including run tracking, history, and run summary sharing. We do not sell personal data and do not use your data for advertising.

Storage and Sync

Data is stored locally on your device by default. If you enable iCloud Sync, run data is synced with your private iCloud account via CloudKit.

Data Rights Requests

You may request access, correction, export, or deletion of your run-related personal data by contacting support@moonlightrun-app.newtree.tech.

Contact

For privacy questions, contact support@moonlightrun-app.newtree.tech.
