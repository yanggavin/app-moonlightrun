MoonlightRun

MoonlightRun helps runners track distance, pace, duration, and route with a focused and readable interface.

Key Features

  • Live run tracking with distance, pace, and duration
  • Run history and detailed summaries
  • Share run highlights to social media with a generated image card
  • Optional iCloud sync for backup

Privacy at a Glance

  • Location permission is required before starting a run
  • Location is used only while an active run is in progress
  • Data stays on your device by default
  • No ads and no sale of personal data
